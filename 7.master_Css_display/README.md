### Block elements such as heading, paragraph or div 
- PROS: height width, transform, rotate and more such properties works fine.
- CONS: Block elements take full line space.

### Inline elements such as anchor tags
- PROS:  Inline elements take ample space.
- CONS: height width, transform, rotate and more such properties doesn't work.
